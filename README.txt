Tela de Login

Uma interface moderna e estilizada de tela de login, desenvolvida em
HTML e CSS, com foco em design limpo, responsivo e intuitivo.

Funcionalidades

-   Campo para usuário (email)
-   Campo para senha
-   Opção “Lembrar senha”
-   Link para recuperar senha
-   Link para cadastro
-   Ícones decorativos usando Boxicons
-   Design transparente com blur (glassmorphism)
-   Totalmente responsivo

Tecnologias Utilizadas

-   HTML5
-   CSS3
-   Google Fonts (Poppins)
-   Boxicons

Estrutura do Projeto

/ |– index.html |– styles.css |– /img └── img.jpg

Instalação

1.  Clone o repositório.
2.  Abra o arquivo index.html no navegador.
